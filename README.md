# Global-Cancer-Trends-and-Patient-Survival-Analysis-2015-2024-
Using Power BI in Healhcare Analytics project to analyze cancer prevalence, treatment cost, risk factors and patients survival trends across multiple countries from 2015-2014.

# Project Overview
This project analyzes global cancer patients data from 2015-2024 using Power BI. The goal is finding trends in cancer incidence, treatment expenses, survival rates, and lifestyle risk factors including alcohol and smoking.

# Business Problem
One of the primary causes of death globally is still cancer. To fully understand patient demographics, treatment costs, and variables linked to cancer severity and survival outcomes, healthcare stakeholders need data-driven insights.

# Objectives
1. Determine which age groups are most impacted.
2. Analyze the prevalence of cancer by location and gender.
3. Analyze the relationship between alcohol use and cancer severity.
4. Analyze smoking habits in relation to cancer types.
5. Compare treatment costs across different cancer types

# Tools Used
1. Power BI
2. Power Query
3. DAX
4. Data Visualization

# Key Findings
1. The highest cancer prevalence was found in patients 65 years of age and older.
2. Severity scores had a positive relationship with alcohol intake level.
3. Severity scores had a positive relationship with alcohol intake level.
4. There were major differences in smoking categories among cancer types.
5. The distribution of patients by gender was relatively equal.
6. The spread of cancer cases across several locations allowed for comparative geographical study.
